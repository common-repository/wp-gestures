== Description ==
This plugin enable a feature that you can use mouse gestures to navigate to the next post or previous post or the home page when on a single post just like mouse gesture in Gmail.

Be aware that this plugin may not work in a browser which internally has mouse gesture capacity.

 
== Installation ==
1. Upload the folder `remind-new` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==
Just like Gmail mouse gesture. Hold your right mouse, move right to skip to the next post, move left to skip to the previous post and move top to return to home.
